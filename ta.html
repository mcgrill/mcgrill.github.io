Tracheal Aire

January 2014

This project was built for the 2014 Society for Technology in Anesthesia (STA) Conference Engineering Challenge. The call to action was to create a system that could use data from MRIs and CT scans to build a customized Williams Intubating Airway for a patient, using only a Makerbot Replicator 2 3D printer.

Our team decided that this custom design could be best implemented in OpenJSCAD, an open source Javascript based CAD program. The final product can produce an airway in two forms, classical and experimental, and can export to four different 3D printing file types.

More information: bit.ly/airwayta

Aspects: 3D Printing, OpenJSCAD

Awards: 1st Place - STA 2014 Engineering Challenge
